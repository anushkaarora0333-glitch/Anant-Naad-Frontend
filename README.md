# Anant Naad

Anant Naad is a full-stack web application designed to help users explore and learn Indian Classical Music through personalized Raga recommendations.

## Features

- Mood-based Raga recommendations
- Searchable Raga database
- Detailed Raga information
- Riyaz Zone with virtual tanpura
- Configurable practice timers
- Responsive user interface

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL
